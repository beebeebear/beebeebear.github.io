——留尾巴很可怕

1.错错错

  • 网文版：

事情发生在一个平常的日子，小编熊需要用一个名为Unity的软件，伊打开搜索引擎，输入并进入了官网 。然而在不断地深入过程中，小编熊发现了一个大秘密……

在小编熊看到以下文字的那一刻，一切都正在改变、崩塌……

【请你遵守以下规则，才能有机会下载到正确的Unity版本。如果期间你看到不干净的东西，请立即、马上卸载包括注册表在内的所有文件，重新进行安装重新进行安装、安装、安、装、安装安装安装安装安装安装————】

2.让我们来一局扫雷

(1)UnityHub吗，有点意思

  • 1)Unity Hub是Unity出的一款管理工具，除了核心的编辑器Editor以外还包括远程服务、社区等综合性功能。所以理论上来说我们只需要下载Unity Hub然后一站式管理。

    • 以下管理工具Hub为3.13版本，编辑器Editor为6000版本，实际为2023的长期版本。

  • 2)安装方式，请注意Unity分为国际版（且2020年后不再对中文区进行支持）和钟区版（特供版，2020年起正式接手），下载时如果你的IP/dns定位在国内，那么即便您通过国际版官网登录，也会强制跳转到钟区版。可以通过下载的安装包的名称进行分辨，钟区版会带上c1/c3等后缀。不同版对应接下来的登录操作具有重大影响，钟区版的账号登录强制要求绑定邮箱和手机号，国际版需要绑定邮箱。

(2)喜欢我时不时重定向的干扰式安装流程吗

  • 1)方案一：在VisualStudio内安装组件（这里提到VisualStudio是因为本次选择了VisualStudio这个编辑器，本质上和IDEA一样就是加强版记事本，非必要但是省力一点）

    • 不太推荐，有可能下载到钟区版，小编暂未测试挂上代理后下载的是什么版本，但未挂代理必然钟区版，以及以安装组件形式安装后续可能不太好管理。

  • 2)方案二：官网下载/三方下载管理工具Unity Hub

    • 国际版官网：https://unity.com
    • 请注意，钟区版官网和国际版极其类似。
    • 无论是官网下载还是三方下载，安装时务必挂好代理，并且开启全局代理，否则必然跳转回钟区版，当然，即便挂好代理也有一定几率跳转回钟区版，一旦发生这种情况，这时请卸载删除UnityHub所有文件（包括注册表包括C:\Users\用户名\AppData\Roaming\目录下的UnityHub文件夹）后重新下载安装。
    • 安装时可以留意左下角的版本来看是否为国际版，起初小编以为沾了中文就不妙但根据结果来看安装包整体中文问题不大，主要还是看版本。当然，这一部分也有可能是因为据热心网友挖掘发现UnityHub这个启动器本质上没有区别，区别在于版本号会涉及到接下来编辑器的版本，国际版仅能下载国际版编辑器，钟区版仅能下载钟区版编辑器。

  • 3)下载期间也别闲着
    • 安装位置建议修改，方便后续管理。
    • 下载期间可以注册一个Unity账号，Unity通过账号管理来管理授权监控收入状况来调整对用户的收费，所以使用Unity都必须登录账号后才能使用（一般来说）。
    • Unity Hub下载完成，这个时候请不要打开启动程序，因为即便挂了全局代理有概率点击Sign in登录后由于官方的重定向等小操作还是会跳转到钟区版登录界面，请注意图中地址的[.cn]后缀。当然，也可以试试看，最新版本全局代理有概率会正常到国际版登录界面。

(3)不.要.放.弃

  • 1)启动脚本
    • 在这里我们使用脚本方式来绕过官方的检测，脚本的使用方式为建立一个文档将以下代码粘贴，再将文件格式改为[.bat]，双击打开，这个时候顺利的话会跳转到国际版的登录界面。
    • 代码如下：

//Windows的代码
@echo off
set HTTP_PROXY=http://127.0.0.1:9910 //这里的127.0.0.1:9910请对应修改成你所用代理的端口号
set HTTPS_PROXY=http://127.0.0.1:9910 //这里的127.0.0.1:9910请对应修改成你所用代理的端口号
start "" "E:\02__NetworkSoftware__Unity\Unity Hub\Unity Hub.exe" //地址根据你安装UnityHub的位置
  • 2)添加许可证
    • 在Setting界面我们可以调整语言和申请许可证，这里的调整语言不会影响到版本请放心，初使用Unity选择Personal授权够用，虽然需要每过三五天重新领取授权。
    • 在这里，有可能会发生点击申请授权结果界面什么都没有，这时可以关闭软件（注意不是退出登录）重新启动软件试试看。

(4)最后一颗雷-下载编辑器

  • 最煎熬的一步来了，注意随时查看全局代理是否仍然开启，编辑器有将近4G大小，不推荐通过Unity Hub内下载（这个时候你肯定会像小编熊一样问那我们下载Unity Hub的意义是？）下载失败概率大且速度慢，可以通过国际版官网内往期版本下载，并且注意，如果期间代理关闭，下载编辑器的网页又会跳转回钟区版，就像世界尽头是淘〇一样。
  • 在这里推荐下载LTS版本，简单来说即长期版本稳定版本，并且选择图中Downloads方式下载而非蓝色标的Install，蓝色标方式意思是跳转回Unity Hub内下载。
  • 如果你平时有用下载管理器，用下载管理器下载即可，如果没有，默认会用浏览器下载，这个时候下载速度有可能也会非常慢以及下载到一半弹出下载失败，如果出现这种状况，推荐下载一个下载管理器，下载管理器一般会有断点下载功能，然后用下载管理器下载。说的比较多的下载器有IDM、Motrix等等，这里小编选择的是FDM下载器，无论是直接托管网页的所有下载链接还是找到下载的直链进行下载都可以。
  • 下载完成后给编辑器一个文件夹。（小编的习惯是Unity一个大文件夹，Hub一个文件夹Editor一个文件夹Project一个文件夹）
